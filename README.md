# Smart Street Lighting System

## 📌 Description
This project presents a smart street lighting system that automatically controls street lights based on ambient light conditions to improve energy efficiency.

## 🎯 Features
- Automatic ON/OFF control based on light intensity  
- Energy-efficient system  
- Simple and low-cost design  
- Simulated using software tools  

## 🛠️ Technologies Used
- LDR Sensor
- LEDs
- Proteus
- Tinkercad

## ⚙️ How It Works
The system uses an LDR (Light Dependent Resistor) to detect surrounding light levels.  
- In daylight: high light intensity → lights OFF  
- In darkness: low light intensity → lights ON  

## 🔌 Components
- LDR Sensor  
- Resistors  
- LEDs  
- Power Supply  

## 🚀 How to Run
1. Open the project in Proteus or Tinkercad  
2. Simulate the circuit  
3. Change light conditions to observe system behavior
